## 역할

app layer --> 은식
tcp / ip layer --> 신영

eth, ni --> 우철

- 함수 base layer 기본 오버라이딩 (카멜 방식)
- 주석 필수 (최대한) 한글로, @param 명시하기
- 변수명은 당연히 영어로 (스네이크)
- 객체지향적 설계를 지향한다 (15줄~ 최대 20줄, 매개변수 최소화)
- 모르면 issue 등록
- 각자 branch --> 클래스 이름 동일
- 레이어 이름으로
